## 🐛 TROUBLESHOOT

**Ảnh bị mờ?**
→ Đã thêm `image-rendering: pixelated` trong CSS

**Font không load?**
→ Kiểm tra internet connection (dùng Press Start 2P từ Google Fonts)

**Mobile layout lỗi?**
→ Test trên Chrome DevTools (F12) → device toolbar

**Form không gửi?**
→ Bình thường (alert hiện thị thôi). Kết nối backend sau.
