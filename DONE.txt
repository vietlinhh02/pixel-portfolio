## 🌐 DEPLOY OPTIONS

### **Netlify** (Easiest, 1 click)
1. Go to netlify.com
2. Drag & drop `pixel-portfolio` folder
3. Done! URL auto-generated

### **GitHub Pages** (Free, versioned)
```bash
git init
git add .
git commit -m "Portfolio"
git remote add origin https://github.com/USERNAME/pixel-portfolio
git push -u origin main
# Enable Pages in settings
```

### **Vercel** (Fast, zero-config)
1. Import GitHub repo or upload folder
2. Auto-deploy on push
3. Done!
