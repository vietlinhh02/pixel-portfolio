# 🎮 PORTFOLIO MỚI - HƯỚNG DẪN NHANH

## ✨ CÓ GÌ MỚI?

### ✅ **Gộp vào 1 Trang**
- Trước: 4 tabs navigation riêng biệt
- Giờ: 1 trang duy nhất, cuộn mượt (smooth scroll)
- Phím tắt: **1-5** để jump nhanh

### ✅ **SVG Icons (Không Emoji)**
- 6 project icons: Game, Web, Art, Chart, Music, Robot
- Avatar: SVG robot character (animated floating)
- Crisp, scalable, professional

### ✅ **Nội Dung Đầy Đủ Hơn**
- Hero: Avatar + Title + CTA + Stats
- Projects: 6 cards với SVG icons, tags, links
- About: Bio + Skills (progress bars) + Timeline + Achievements
- Stats: 4 big numbers + 3 testimonials
- Contact: Form + Info + Social links

## 🔧 TUỲ CHỈNH MÀU

Mở `css/style.css`, find `:root`, đổi:

```css
:root {
    --primary: #0b3d91;      /* Dark blue → your color */
    --secondary: #08a4de;    /* Cyan → your color */
    --accent: #ffd700;       /* Gold → your color */
    --dark: #1a1a1a;
    --light: #f0f0f0;
    --danger: #e74c3c;
    --success: #27ae60;
    --warning: #f39c12;
}
```

**Gợi ý palette:**
- Retro NES: `#c41e3a`, `#4B53B6`, `#FFB900`
- Gameboy: `#0F380F`, `#306230`, `#8BAC0F`, `#9BBC0F`
- Cyberpunk: `#FF006E`, `#8338EC`, `#3A86FF`, `#FB5607`
- Pastel: `#FFB3BA`, `#FFDFBA`, `#FFFFBA`, `#BAE1FF`

## ✏️ CHỈNH ICON

SVG icons được embed trực tiếp trong HTML. Để đổi:

1. **Mở**: `index.html`
2. **Tìm**: `<svg viewBox="0 0 64 64"...` (trong project cards)
3. **Edit**: Đường stroke/fill hoặc thay svg khác
4. **Lưu** và F5 refresh

## 🐛 COMMON ISSUES

**Q: Icons mờ?**  
A: SVG đã set `image-rendering: pixelated` - check browser zoom

**Q: Form không gửi?**  
A: Normal - alert hiện thị. Để gửi email: dùng Formspree hoặc Emailjs

**Q: Mobile layout cấp?**  
A: Check responsive CSS media queries trong `style.css`

**Q: Font không load?**  
A: "Press Start 2P" từ Google Fonts. Check internet hoặc fallback monospace
